### doc文档仓库目录分类标准

1. 根据技术种类分类，包括框架

2. 工作中采过的坑集中在 - 爬坑记录

3. 微信相关的集中在 - 微信 （公众号开发，js-sdk，小程序等）

4. 谷歌浏览器 chrome 扩展开发集中在 - chrome扩展

5. 有关IDE，git等辅助编写代码的集中安置在 - IDE

6. 关于第三方库的使用集中放置在 - 库

7. 库 - 下面再进行技术分类


** icode仓库目录尽量与doc一致 **
